# flag-icons [![Build Status](https://travis-ci.org/Collaborne/iron-flag-icons.svg?branch=master)](https://travis-ci.org/Collaborne/iron-flag-icons)

Polymer icon set for flags

To use this element:

`bower install iron-flag-icons`
